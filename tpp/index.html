Traci’s Purple Presser

View on GitHub

Traci’s Purple Presser is a simple PyGame project I built over a weekend on my phone using Pydroid3. The version on this page was ported with Pygbag. I got the idea from my wife’s cousin’s physical therapy sessions, where she practiced pressing colored buttons to build coordination. I designed the game with randomly generated purple buttons, confetti explosion, and a popping sound. It ended up giving her endless amounts of simple fun.
